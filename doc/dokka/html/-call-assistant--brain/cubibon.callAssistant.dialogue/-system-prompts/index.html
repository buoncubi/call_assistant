SystemPrompts

data class SystemPrompts(val promptsManager: PromptsManager, val callSections: List<String>, val summarySections: List<String>)

A data class that holds the configuration for system prompts used in a dialogue.

It bundles together the manager for all prompt templates and specifies which sections of those templates should be used for the main conversation (callSections) and for generating summaries (summarySections).
