FieldsMatcher

data class FieldsMatcher<P>(val fields: P, val ranges: List<IntRange>)

Represents a data structure that contains string matched in a text and the relative starting and ending indexes where such a match occurred in the prompts special sections.

It is used in PromptsParser.parseSpecialSections with T: FieldsMap and in PromptsParser.extractSpecialSection with T: List<String>.
