PromptsManager

abstract class PromptsManager : TypedPrompts

A general implementation of a class that can be used to exploit prompt representation at runtime in an efficient manner.

This class defines the applyVariables, which is charge to create the prompts based on the rawPrompts and the variables maps. This function has the effect to replace variables' placeholders in the prompts whit values computed by functions defined in VariablesFunction.

Instance of this class are given by the PromptsDeserializer and PromptsParser.
