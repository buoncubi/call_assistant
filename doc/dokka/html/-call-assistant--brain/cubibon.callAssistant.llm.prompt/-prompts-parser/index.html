PromptsParser

object PromptsParser : Loggable

Parse a specific syntax to define prompts to guide LLM model; such a syntax is documented in a LLM_PROMPT-LLM_MESSAGE-README.md file placed in the same directory of this class.

This class provides the parse method, which returns an instance of ParsedPrompts. The latter can be serialized to file and deserialized through PromptsDeserializer. It is recommended to use this class only offline since it might be time-consuming. At runtime, the prompt data can be deserialized to get an implementation of the PromptsManager, which can be efficiently used.
