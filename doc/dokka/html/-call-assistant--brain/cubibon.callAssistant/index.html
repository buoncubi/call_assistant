CallbackInput

interface CallbackInput

The base definition of the input to the callback managed by CallbackManager. It only defines a property that is neither processed by the CallbackManager nor by ServiceInterface, but it is propagated in such a way that the classes that exploit the functionalities of a ServiceInterface (thus, might generate callbacks), can pass data to the callbacks.

CallbackManagerInterface

interface CallbackManagerInterface<I : CallbackInput>

The base definition of a callbacks manager, which collects lambda functions in a map, and allow invoking them. This class should store lambda functions on map with a string-based identifier for each callback. Such a manager is usually used by an implementation or extension of the ServiceInterface.
