MetaData

interface MetaData

The message metadata container. It is associated with a message exchange between the LLM-based assistant and the user by MessageWrapper. The metadata it represents will be stored in a NoSQL database for logging purposes (e.g., on AWS DynamoDB), and the implementation of this interface can further extend the stored data.

Note that the properties below might occur in the map returned by toMap if they are not null.
