MetaRole

enum MetaRole : Enum<MetaRole>

Enumeration of possible message types in the conversation, which can be:

  • USER, when the message is sent by the user;

  • ASSISTANT, when the message is sent by the assistant;

  • SUMMARY, when the message is a summary of the previous conversation (see MessagesManager for more info about the summarization procedure).

Note that the MessageWrapper.role returned by MetaData.toMap are the lowercase of the enum's name with the camelcase standard; as returned by toString.
