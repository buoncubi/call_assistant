Summarizing

data class Summarizing(val messages: List<Summarizing.SummarizingMessage>, val previousSummary: Summarizing.SummarizingMessage? = null)

The data class used to represent the messages to be summarized. A list of this class it encoded in Summarizing, given by MessagesManager.getSummaryInfo, and required by MessagesManager.addSummary. It uses Summarizing.format to format the messages to be summarized.
