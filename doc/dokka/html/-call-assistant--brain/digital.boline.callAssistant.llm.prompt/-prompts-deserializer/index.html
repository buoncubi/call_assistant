PromptsDeserializer

object PromptsDeserializer : Loggable

An object that can be used to deserialize ParsedPrompts from file, either from a Json or Binary format.

This class should be used at runtime to have access to PromptsManager after that a ParsedPrompts has been generated by PromptsParser and serialized into a file.
