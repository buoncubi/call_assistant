CallbackManager

class CallbackManager<I, O>(logger: CentralizedLogger) : CallbackManagerInterface<I, O>

An implementation of the CallbackManagerInterface that collects lambda functions in a map where keys identifies callback based on where they are implemented, i.e., based on getCallbackIdentifier. This class is usually used by an implementation or extension of the ServiceInterface.
